# 开发者

> 本项目基于quasar开发，最好全局安装quasar脚手架。

## 安装依赖

```bash
yarn
# or
npm install
```

### 启动项目

```bash
quasar dev
```

### 构建生产版本

```bash
quasar build
```

### 自定义配置

查看 [Configuring quasar.config.js](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js).
